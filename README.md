# Rahul Portfolio

Personal portfolio site built with Next.js 15, Tailwind CSS v4, and Motion.

## Dev

```bash
npm install
npm run dev
```

## Customize

- `app/header.tsx` -- name and title
- `app/data.ts` -- projects, work experience, blog posts, social links, email
- `app/page.tsx` -- intro bio
- `app/blog/<slug>/page.mdx` -- blog posts
